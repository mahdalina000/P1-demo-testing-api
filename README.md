# Demo Testing API menggunakan Postman

Praktikum ini bertujuan untuk memahami pengujian API menggunakan Postman
dengan metode GET, POST, PUT, dan DELETE.

## Tools
- Postman
- API: JSONPlaceholder

## Endpoint yang Diuji
- GET /posts
- GET /posts/{id}
- GET /comments?postId=1
- POST /posts
- PUT /posts/{id}
- DELETE /posts/{id}

## Hasil
Seluruh request berhasil dijalankan dan menghasilkan response
dalam format JSON sesuai dengan dokumentasi API.
